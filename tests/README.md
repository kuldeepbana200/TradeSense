# Test Suite Overview

This directory contains the test suite for the statistical arbitrage platform.

- `pytest.ini` – Configuration for pytest (test paths, markers).
- `UNIT_TEST_COMPLETION_SUMMARY.md` – Summary of test targets and coverage expectations.
- `conftest.py` – Shared fixtures for unit and integration tests.

Run the test suite locally using `pytest` and consult `UNIT_TEST_COMPLETION_SUMMARY.md` for guidance on critical tests to verify before release.
